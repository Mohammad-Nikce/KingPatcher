# 👑 King Patcher  
**نسخه‌ای شخصی‌سازی‌شده از Lucky Patcher برای برنامه‌های ایرانی (مایکت و بازار)**

---

## 📌 معرفی

King Patcher یک ابزار رایگان و آزمایشی است که به شما کمک می‌کند **بخش‌های پولی و ویژه‌ی بازی‌ها و برنامه‌های ایرانی را بدون پرداخت هیچ هزینه‌ای فعال کنید**.  
این برنامه مخصوص برنامه‌هایی است که از فروشگاه‌های **مایکت** و **بازار** نصب کرده‌اید.  
این پروژه فقط برای هدف‌های آموزشی و پژوهشی ساخته شده تا علاقه‌مندان با شیوه‌ی کار سرویس‌های پرداخت درون‌برنامه‌ای آشنا شوند.

---

## ⚠️ هشدار مهم (قانونی و اخلاقی)

- فروش یا استفاده‌ی تجاری از این برنامه **ممنوع** است.  
- استفاده از این ابزار ممکن است باعث **مسدود شدن حساب کاربری** شما در بازی‌ها و برنامه‌ها شود.

---

## 🚀 ویژگی‌ها

- فعال‌سازی رایگان خریدهای درون‌برنامه‌ای در برنامه‌های **مایکت**  
- فعال‌سازی رایگان خریدهای درون‌برنامه‌ای در برنامه‌های **بازار**  
- قابل استفاده روی گوشی‌های **بدون روت** (با انجام تنظیمات ساده)

## ⚙️ نکاتی که بهتر است بدانید

- **گوشی‌های روت‌شده:** برنامه روی این گوشی‌ها بهتر کار می‌کند و معمولاً نیازی به نصب دوباره‌ی برنامه‌ی تغییرداده‌شده ندارید.  
- **گوشی‌های بدون روت:** ممکن است لازم باشد نسخه‌ی اصلاح‌شده‌ی برنامه را به‌جای نسخه‌ی اصلی نصب کنید.  
- این ابزار روی بازی‌های آنلاین که مدام به اینترنت وصل هستند (مثل کلش آو کلنز) جواب نمی‌دهد و حتی ممکن است باعث **هنگ کردن یا بسته شدن ناگهانی** برنامه شود.
